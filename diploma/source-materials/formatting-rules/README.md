# Правила оформления

Сюда складываются PDF или DOCX с формальными требованиями кафедры,
университета и нормоконтроля.

## Текущий набор материалов

Текущий набор материалов:

- `общие_требования_к_учебным_документам.pdf`
- `правила_оформления_источников.pdf`
- `правила_оформления_перечней.pdf`
- `форматирование.pdf`

При расхождении между черновым текстом и оформлением источников приоритет
отдаётся официальным правилам из PDF и примеру прошедшей нормоконтроль ВКР.
