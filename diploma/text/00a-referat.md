# Реферат

Выпускная квалификационная работа бакалавра содержит 4 рисунка, 8 таблиц и 30 использованных источников. (источник: diploma/text/00b-abbreviations.md; diploma/text/02-review.md; diploma/text/04-architecture.md; diploma/text/06-experimental-design.md; diploma/text/07-experiments.md; diploma/text/09-references.md)

Ключевые слова: ДЕТЕКЦИЯ МАЛЫХ ОБЪЕКТОВ, АУГМЕНТАЦИЯ, ADAPTIVE POLICY, COCOEVAL, VISDRONE, YOLO, ALBUMENTATIONS, OBJECT BANK. (источник: README.md; diploma/docs/narrative.md; diploma/docs/glossary.md)

Тема работы - разработка воспроизводимого конвейера адаптивного подбора аугментаций для детекции малых объектов на основе анализа структуры датасета, интерпретируемого механизма правил и экспериментальной оценки через COCOeval. (источник: README.md; docs/AUGMENTATION_POLICY.md; src/pipeline_mvp.py; diploma/docs/narrative.md)

## Источники раздела

- `diploma/template/blank-template.docx`. Использован для согласования структуры реферата с шаблоном итогового документа. (источник: diploma/template/blank-template.docx)
- `diploma/source-materials/examples/ВКРБ_МаршунинаМА.docx`. Использован как пример размещения и состава реферата. (источник: diploma/source-materials/examples/ВКРБ_МаршунинаМА.docx)
- `README.md`. Использован для формулировки темы работы и описания состава конвейера. (источник: README.md)
- `diploma/docs/narrative.md`. Использован для согласования реферата с общей логикой ВКР. (источник: diploma/docs/narrative.md)
- `diploma/docs/glossary.md`. Использован для унификации терминологии и ключевых слов. (источник: diploma/docs/glossary.md)
- `docs/AUGMENTATION_POLICY.md`. Использован для описания rule-based части подхода. (источник: docs/AUGMENTATION_POLICY.md)
- `src/pipeline_mvp.py`. Использован для подтверждения состава основного программного конвейера. (источник: src/pipeline_mvp.py)
