# Журнал решений

В данном файле фиксируются редакторские, структурные и оформительские решения по ВКР, которые влияют на состав текста, правила цитирования и подготовку итогового Word-документа.

- как зафиксированы названия глав;
- какие рисунки и таблицы вошли в финальную версию;
- какие формулировки источников подтверждены примерами `.docx`;
- какие разделы были сокращены после первой сборки в Word.
