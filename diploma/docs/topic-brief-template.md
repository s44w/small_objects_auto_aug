# Шаблон тематического пакета для написания раздела

Этот шаблон нужен для полуавтоматической постановки задачи на написание отдельной темы, подраздела или главы. Чем точнее заполнен пакет, тем быстрее и стабильнее можно готовить текст, таблицы, рисунки и формулы.

## Как использовать

1. Скопировать шаблон для новой темы.
2. Заполнить обязательные поля.
3. Положить рядом список файлов, источников и материалов.
4. После этого запускать написание конкретного раздела.

## Шаблон

```md
# Тема

## Выходной файл

- `diploma/text/...`

## Место в структуре ВКР

- глава:
- подраздел:
- связь с предыдущим разделом:
- связь со следующим разделом:

## Цель фрагмента

- что именно должен доказать или объяснить данный текст;
- какие тезисы должны остаться у читателя после прочтения.

## Целевой объем

- минимальный объем:
- целевой объем:
- максимальный объем:

## Обязательные источники

- [ ] `[N]` ...
- [ ] `[N]` ...
- [ ] локальный источник: `src/...`
- [ ] локальный источник: `docs/...`
- [ ] URL для каждого внешнего источника: `https://...`

## Дополнительные источники

- [ ] ...

## Ключевые тезисы

- ...
- ...
- ...

## Обязательные рисунки

- [ ] рисунок:
  - источник:
  - что именно показывает:
  - куда вставить:
  - нужен ли перевод подписи:

## Обязательные таблицы

- [ ] таблица:
  - источник:
  - что сравнивает:
  - какие столбцы обязательны:
  - куда вставить:

## Обязательные формулы

- [ ] формула:
  - источник:
  - обозначения:
  - зачем нужна в тексте:
  - куда вставить:

## Локальные материалы проекта

- [ ] `src/...`
- [ ] `configs/...`
- [ ] `docs/...`
- [ ] `tests/...`
- [ ] `notebooks/...`

## Ограничения

- чего нельзя утверждать без дополнительной проверки;
- какие результаты пока отсутствуют;
- какие формулировки должны быть осторожными.

## Итоговый контроль

- [ ] после каждого абзаца есть ссылка;
- [ ] рисунки имеют ссылки на источники;
- [ ] таблицы имеют ссылки на источники;
- [ ] формулы имеют ссылки на источники;
- [ ] у каждого внешнего источника указан полный URL;
- [ ] термины согласованы с `diploma/docs/glossary.md`;
- [ ] структура согласована с `diploma/docs/narrative.md`;
- [ ] оформление согласовано с `diploma/docs/style-guide.md`.
```
